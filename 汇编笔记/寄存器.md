# 寄存器

## 寄存器分类

+ 通用寄存器
  + AX
  + BX
  + CX
  + DX
  + DI
  + SI
+ 特殊寄存器
  + SP(栈指针)
  + BP(栈基址)

## 寄存器大小

+ _XX_ (e.g. AX,SP) -> 16 位
+ E _XX_ (e.g. EAX,ESP) -> 32 位
+ R _XX_ (e.g. RAX,RSP) -> 64 位
