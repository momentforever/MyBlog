# 优雅的断开套接字的连接

```cpp
#include <sys/socket.h>
int shutdown(int sock,int howto);
/*
成功时返回 0，失败时返回 -1
sock:需要断开套接字文件描述符
howto:传递断开方式信息
*/
```
``howto``参数选择：
+ ``SHUT_RD``:断开输入流
+ ``SHUT_WR``:断开输出流
+ ``SHUT_RDWR``:同时断开I/O流
